Gráficos dinámicos

Este proyecto busca ofrecer una alternativa interactiva y llamativa para la presentación de datos para los que comunmente se usan hojas de cálculo comerciales y dashboards.

Objetivo

Visualización efectiva


Presentación precisa de datos en un formato visual muy agradable y de interacción amigable

Beneficios

Personalización 

Selección de personalizada de datos a ser representados con cinco estilos diferentes de gráficas

Sobre el proyecto

Presentación de datos

Una parte importante para la efectiva transmisión y entendimiento de la información reside en la presentación de la misma

Con este pensamiento en mente se desarrollo esta herramienta a fin de facilitar al gestor de datos una elegante presentación de los mismos.

Fuentes de información

De momento la aplicación solo puede graficar el contenido de un archivo CSV subido a la aplicación por el usuario.

Se tiene un espacio reservado a la subida de archivos locales. Favor asegurarse que el archivo sea un CSV delimitado por comas, de lo contrario no se podrá graficar.

Gráficas disponibles

La presente aplicación cuenta con 5 tipos de vistas para los datos. Tenga en cuenta que algunas de las vistas no son compatibles con algunos sets de datos.

Scatter

La gráfica de dispersión requiere de hasta 4 pares de datos a ser introducidos. Cada par corresponde a dos columnas cuyas filas sean numeros. 

Este tipo de gráficos es ideal para visualizar la relación de 2 variables numerales.

Line

Se cumplen los mismo requerimientos que para el gráfico scatter con respecto a los datos requeridos.

La utilidad de este tipo de gráficos puede encontrarse para datos que representen una evolución en el tiempo. Otra utilidad menos común puede ser para graficar lineas con valores X e Y.

Barras

Se requiere una columna que contenga las etiquetas de las barras a representar. En conjunto, se pueden seleccionar un número arbitrario de columnas numéricas.

Se tiene la opción de presentar las barras de manera horizontal o vertical, según se vea conveniente. Este tipo de gráficos es ideal para representar conjuntos con valores cuantitativos, al permitirse más de un conjunto de valores, se pueden visualizar las misma variables en diferentes contextos (ej. Ventas de los mismos productos en 2 meses diferentes).

Pie

Es requerido una columna con las etiquetas y otra con valores cuantitativos, este es el Tipo de gráfico que menos columnas necesita.

Este tipo de gráficos permite ver las proporciones de diferentes variables en un mismo contexto.

Sobre las tecnologías

Para el presente se emplearon variedad de librerías y herramientas referentes al desarrollo web.


Para el front-end se utilizó Ajax en conjunto con HTML5, CSS y Javascript. Para el backend se empleó Python a través del Framework Flask y para la generación de gráficos se empleó la librería Pyecharts
